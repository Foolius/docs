# `:open  %~  "censig"`

`{$cnsg p/wing q/seed r/seed}`: call with multi-armed door.

```

```

## Syntax

Regular: *3-fixed*.

Irregular: `~(a b c)` is `%~(a b c)`; `~(a b c d e)` is `%~(a b
[c d e])`.
